Installation

To install arraylib-solve first create Python 3.8 environment e.g. by

conda create --name arraylib-env python=3.8
conda activate arraylib-env

and install the package using

pip install arraylib-solve

arraylib-solve uses bowtie2 [1] to align reads to the reference genome. Please ensure that bowtie2 is installed in your environment by running:

conda install -c bioconda bowtie2

[1] Langmead, B. and Salzberg, S.L., 2012. Fast gapped-read alignment with Bowtie 2. Nature methods, 9(4), pp.357-359.
